#  Turbine Health Monitoring & Failure Prediction System

##  Project Overview  
This project focuses on analyzing turbine operational data to predict equipment health and identify potential failure types using machine learning techniques.

It follows a structured data workflow, transforming raw sensor data into meaningful insights and predictive models that support proactive maintenance.

---

##  Objectives  
- Clean and preprocess turbine sensor data  
- Perform exploratory data analysis (EDA)  
- Identify patterns influencing turbine performance  
- Build predictive models for health classification  
- Detect and categorize turbine faults  

---

##  Tools & Technologies Used  
- Python – Pandas, NumPy  
- Data Visualization – Matplotlib, Seaborn  
- Machine Learning – Scikit-learn, XGBoost  
- Jupyter Notebook  
- CSV Dataset  

---

##  Analysis Performed  
- Data cleaning and preprocessing  
- Handling missing values and outliers  
- Exploratory Data Analysis (EDA)  
- Feature selection and correlation analysis  
- Model building using:
  - Logistic Regression  
  - Decision Tree Classifier  
  - Random Forest Classifier  
  - XGBoost Classifier  
- Model evaluation using accuracy, confusion matrix, and classification report  

---

##  Key Insights  
- Vibration and temperature are key indicators of turbine failures  
- Ensemble models like Random Forest and XGBoost performed better  
- Strong relationships exist between certain features and fault types  
- Early prediction can help reduce downtime and maintenance costs  

---

## 📈 Project Outcome  
This project demonstrates an end-to-end machine learning pipeline:

Raw Data → Cleaning → EDA → Model Building → Evaluation → Insights  

It reflects practical skills required for data analytics and machine learning roles, including data processing, modeling, and interpretation.
